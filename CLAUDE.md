# 마음 코치 프로젝트

## 기술 스택
- HTML + CSS + JavaScript (단일 페이지 앱)
- Anthropic Claude API (claude-sonnet-4-20250514)
- localStorage로 데이터 저장
- 서버 없음, 프론트엔드만
- 배포: GitHub Pages 또는 Vercel (무료)

## 참고 문서
- 전체 기획: docs/마음코치_최종기획안_v5.md
- 개발 브리핑: docs/마음코치_개발브리핑_v2.md

## 개발 우선순위
- P0: 온보딩 → 매일 기록 → 주간 리포트(LLM) → 세션 브리핑 토글(LLM) → 정량 데이터 계산
- P1: AI 대화(공감/코칭) → 음성 입력 → 감정 차트 → 데모 데이터
- P2: 응원/오늘의질문 → 타임라인 → 다크모드 → 내보내기

## 규칙
- 유저 대면 텍스트에서 임상 전문 용어(인지 왜곡, 파국화, 흑백사고 등) 사용 금지
- 상담사 뷰에서만 전문 용어 허용
- 상담사 뷰 하단에 AI 디스클레이머 필수
- 정량 데이터(이행률, 빈도)는 JS로 직접 계산 — LLM에 맡기지 않음
- API 키를 코드에 하드코딩 금지 — 설정 화면에서 입력받아 localStorage 저장
- 모바일 퍼스트 (최대 너비 680px)
- 디자인: 보라(#7c6fcd) 기본, 초록(#52b788) 성공, 로즈(#e07b9a) 감정
