# 마음 코치 — 기획안 v3

## 한 줄 요약
> "오늘 어땠는지 말해보세요. AI가 점점 더 깊이 나를 분석해줍니다. 쓸수록 정확해진다."

---

## 타겟
- 고민 많은 일반인 (20~30대 직장인 중심)
- 상담을 받을지 말지 고민하는 사람
- 특별한 문제가 없어도 자기 이해에 관심 있는 사람

---

| | 파인드마이와이 | 일반 감정일기앱 | **마음코치** |
|--|--------------|---------------|------------|
| 기록 방식 | 텍스트 답변 | 텍스트 타이핑 | **음성으로 말하기** |

**핵심 메시지:** "타이핑 말고 그냥 말하세요. 쓸수록 나를 더 잘 아는 AI"

## 두 기능의 차별화

| | 🎙️ 마음 기록 | 💬 마음 대화 |
|--|------------|------------|
| 입력 방식 | **음성** (말하기) | 텍스트 (채팅) |
| 느낌 | 일기 쓰듯 혼자 털어놓기 | AI와 주고받기 |
| 목적 | 하루 감정 기록 → 패턴 분석 재료 | 실시간 공감/코칭 대화 |
| 분석 활용 | 주간 리포트 / 심층 분석 데이터 | 즉각적인 감정 반응 |

## 핵심 구조: 단계별 잠금 해제

```
가입 ──→ 초기 설문 ──→ Lv.1 기본 프로필
              │
      🎙️ 매일 음성 기록 / 💬 마음 대화
              │
         ┌────┴────┐
      3일째     7일째
    Lv.2        Lv.3
  표현 분석   첫 주간 리포트
              │
         ┌────┴────┐
   14일+대화3회  21일+대화5회
    Lv.4          Lv.5
  심화 리포트    ★ 심층 분석 ★
                    │
              (문제 감지 시)
                    │
            전문가 상담 안내
```

| 단계 | 조건 | 기능 |
|------|------|------|
| **Lv.1** | 가입 즉시 | 초기 프로필 리포트 |
| **Lv.2** | 기록 3일 | 자주 쓰는 표현 + 감정 어휘력 분석 |
| **Lv.3** | 기록 7일 | 첫 주간 리포트 + 감정 에너지 맵 |
| **Lv.4** | 기록 14일 + 마음 대화 3회 | 심화 리포트 (변화 카드 + 번아웃 체크) |
| **Lv.5** | 기록 21일 + 마음 대화 5회 | 심층 분석 (초기 프로필 비교 + 전문가 안내) |

**잠금 해제 UX:** 미해제 기능은 누르면 "기록 OO일 + 마음 대화 OO회 달성 시 이용 가능" + 진행률 바 표시. 숨기지 않고 보여줘서 동기 유발.

## 화면 구성

### ① 🎙️ 마음 기록 (홈)

**첫 실행 — 초기 프로필 설문 (10~15개)**
- 감정 인식: "슬플 때 보통 어떻게 반응해요?"
- 스트레스 대처: "압박감 느끼면 주로?"
- 대인관계: "갈등이 생기면 나는?"
- 자기 인식: "나의 강점/약점은?"
- 에너지: "하루 중 가장 힘든 시간대는?"
- 목표: "이 앱에서 뭘 얻고 싶어요?"

→ 결과: Lv.1 기본 프로필 리포트

**매일 — 음성 기록 화면**
- 큼직한 마이크 버튼 (중앙)
- "오늘 어땠는지 말해보세요" 안내 문구
- 누르면 음성 인식 시작 → 텍스트로 변환되어 저장
- 텍스트 입력도 가능 (음성이 불편한 환경 대비)
- 저장 후 간단한 AI 피드백 카드 ("오늘도 기록했어요 +1일")
- 진행률 바: "심층 분석까지 5일 남았어요!"

**구현:**
- 데모: Web Speech API (브라우저 내장, 무료) → 음성→텍스트
- 장기: Gemini Audio API → 음성 톤/속도까지 감정 분석 (P3)

### ② 💬 마음 대화

**공감 모드**
- 들어주기 중심, 해결책 안 줌
- "그랬구나. 힘들었겠다."

**코칭 모드**
- 리프레이밍 질문
- "정말 '항상' 그래요? 최근에 잘 된 일은 없었나요?"
- 초기 프로필 결과 참고해서 개인화 코칭

**오늘의 질문 (P2)**
- 심리학 기반 셀프 리플렉션
- "오늘 가장 감사했던 순간은?"

### ③ 📊 리포트

단계에 따라 내용이 달라짐:

**Lv.1 — 초기 프로필 (가입 직후)**
```
📊 나의 마음 프로필

🧠 감정 처리 스타일: 내면화형
⚡ 스트레스 반응: 회피형
💬 대인관계: 조화 추구형
🌱 맞춤 추천: 코칭 모드로 감정 표현 연습 추천
```

**Lv.2 — 표현 분석 (3일차~)**
```
💬 자주 쓰는 표현: "항상" 4회 · "또" 3회
📝 감정 어휘력: 8단어
```

**Lv.3 — 첫 주간 리포트 (7일차~)**
```
💬 이번 주 감정 흐름 요약
🗺️ 감정 에너지 맵 (시각화)
📝 감정 어휘력 성장 (↑4단어)
🌱 긍정 포인트
→ 다음 주도 기록하면 변화를 비교해드릴게요!
```

**Lv.4 — 심화 리포트 (14일+대화3회~)**
```
🔄 변화 카드
  지난 주: "실수했는데 뭐 괜찮아"
  이번 주: "또 실수했다. 나는 항상 이런 식이야"
  → 같은 상황인데 자책이 강해지고 있어요.

📊 주간 비교 수치
  부정 표현: 8회 → 14회 (↑75%)

🧠 감지된 사고 습관 (일상 언어)
  "안 좋은 일이 생기면 최악을 먼저 떠올리는 패턴 3회"

🔥 번아웃 체크
  현재: 2단계 (스트레스 축적기)
  ████░░ 지금이 관리하기 좋은 타이밍이에요!

💡 맞춤 제안 3가지
→ 심층 분석까지 7일!
```

**Lv.5 — 심층 분석 (21일+대화5회~)**
```
📊 3주간 심층 분석 리포트

🧠 사고 패턴 심층 분석 (Top 3)
📈 3주 변화 흐름 차트
📝 감정 어휘력 성장 (8→18단어)
🔥 번아웃 추이 (주차별)
🔬 초기 프로필과 비교
  설문: "스트레스에 강한 편"
  실제: 스트레스 상황에서 회피+자책 패턴 반복
  → 이 차이 자체가 중요한 발견이에요.
💡 종합 맞춤 전략

⚠️ 필요시: "전문가와 이야기해보는 건 어떨까요?"
  📞 정신건강위기상담전화 1577-0199
  ※ AI 생성 참고자료, 전문 진단 대체 아님
```

### ④ 📅 나의 기록
- 날짜별 기록 모아보기
- 음성 기록 재생 (장기 기능)
- 감정 키워드 태그

### ⑤ ⚙️ 설정 (모달)
- Gemini API 키 입력
- 테스트 모드 ON/OFF
- 초기 프로필 다시 하기
- 데모 데이터 불러오기 (3주치)
- 다크모드 (P2)
- 데이터 초기화

## 심리 분석 기법 3가지

### 1. 감정 에너지 맵
감정을 "에너지 높음/낮음" × "기분 좋음/나쁨" 2축으로 매핑.
매주 점이 어디에 찍히는지 보면 변화가 한눈에 보임.
- 적용: Lv.3부터

### 2. 감정 어휘력 분석
"힘들다"만 반복하는지, "억울하다/서운하다"를 구분해서 쓰는지.
단어 수 증가 자체가 성장의 증거.
- 적용: Lv.2부터

### 3. 번아웃 단계 체크 (5단계)
기록 변화 추이로 현재 번아웃 단계 추정.
타겟(직장인)에게 직접적으로 와닿음.
- 적용: Lv.4부터

## 개발 우선순위

### P0 (데모 필수)
1. 초기 프로필 설문 + Lv.1 리포트
2. 🎙️ 음성 기록 화면 (Web Speech API)
3. 단계별 잠금 해제 + 진행률 바 ← 현재 구현됨
4. Lv.3 주간 리포트 (감정 에너지 맵 + 감정 어휘력)
5. Lv.4 심화 리포트 (변화 카드 + 번아웃 체크)
6. Lv.5 심층 분석 (초기 프로필 비교 + 전문가 안내)
7. IS_TEST_MODE + 데모 데이터 3주치

### P1 (데모 퀄리티)
8. 💬 마음 대화 (공감/코칭 모드) - Gemini API 연결
9. 감정 에너지 맵 차트 시각화
10. Vercel 배포

### P2 (시간 남으면)
11. 오늘의 질문
12. 다크모드
13. 나의 기록 탭

### P3 (장기)
14. Gemini Audio API - 음성 톤/속도 감정 분석

## 데모 시나리오 (5분)

### 1분 — 인트로 + 초기 설문
> "타이핑 말고 그냥 말하세요."
> 초기 설문 → Lv.1 프로필 리포트

### 1분 — 음성 기록 시연
> 마이크 버튼 누르고 말하면 → 텍스트 변환 → 저장
> "이게 매일의 기록이 돼요"

### 1분 — 단계별 잠금 해제
> "3일 쓰면 이게 열리고, 7일이면 이게 열려요"
> 데모 데이터로 빠르게 Lv.3~4 보여줌

### 1.5분 — Lv.4~5 리포트 (하이라이트)
> 변화 카드 + 번아웃 체크 + 초기 프로필 비교

### 0.5분 — 마음 대화
> 코칭 모드 짧게 시연
