# **App Name**: ReguMate

## Core Features:

- Regulation Knowledge Base: Upload, view, and manage enterprise regulations and policies in a centralized repository. This includes functionality to update existing documents and remove outdated ones.
- AI Regulation Revision Drafter: An AI tool that generates recommended revisions for existing company regulations by processing new legal directives or internal mandates provided by the user, suggesting changes to ensure compliance.
- AI Justification Extractor: An AI tool that summarizes drafted regulations by identifying the limitations of previous rules, outlining the reasons for proposed revisions, and highlighting the key changes for executive review.
- AI Compliance Chatbot: An AI tool that provides instant answers to user questions about company policies, compliance, and internal regulations by leveraging information from the knowledge base.
- AI Interaction Settings: Provides controls for how the AI processes information, including a toggle for a strict privacy mode (preventing AI training on input data) and an adjustable setting for vector database retrieval strictness.

## Style Guidelines:

- The primary action color is a deep, professional blue (#2360D4), signifying trust and authority, ideal for main buttons and interactive elements.
- The background color for main content areas is a subtle off-white (#F8FAFC), providing a clean and readable canvas.
- A muted greenish-blue (#99C2C9) serves as the accent color, offering gentle visual separation and highlighting for secondary UI elements.
- Headlines utilize 'Space Grotesk' (sans-serif) for a modern, tech-forward, and assertive feel.
- Body text is set in 'Inter' (sans-serif), chosen for its excellent readability and neutral, objective aesthetic.
- References to specific document names or technical snippets are distinguished with 'Source Code Pro' (monospace), improving clarity and consistency with the data context.
- A consistent set of clean, modern, and outline-style icons are used throughout the application to clearly represent functions and concepts (e.g., Lucide icons).
- The application features a structured two-column layout with a fixed left sidebar for navigation and global settings, complemented by a fixed top header for branding and global actions. Main content areas are fluid but width-constrained for optimal readability.
- Subtle and functional UI animations are used for element transitions, hover states, and input feedback, ensuring a smooth and responsive user experience without unnecessary distractions.
