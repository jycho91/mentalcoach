# 마음 코치 — 기획안 v3

## 한 줄 요약
> "매일 감정을 기록하면 AI가 점점 더 깊이 나를 분석해주는 앱. 쓸수록 정확해진다."

---

## 타겟
- 고민 많은 일반인 (20~30대 직장인 중심)
- 상담을 받을지 말지 고민하는 사람
- 특별한 문제가 없어도 자기 이해에 관심 있는 사람

---

## 포지셔닝

| | 파인드마이와이 | 일반 감정일기앱 | **마음코치** |
|--|--------------|---------------|------------|
| 분석 깊이 | 깊음 (1회) | 없음 | **점점 깊어짐** |
| 사용 빈도 | 1회성 | 매일 (기록만) | 매일 (기록+대화+분석) |
| 변화 추적 | 불가 | 불가 | **매주 추적** |
| 가격 | 3~10만원 | 무료 | 무료 |

**핵심 메시지:** "한 번 분석하고 끝나는 게 아니라, 쓸수록 나를 더 잘 아는 AI"

## 핵심 구조: 단계별 잠금 해제

```
가입 ──→ 초기 설문 ──→ Lv.1 기본 프로필
              │
         매일 기록 / AI 대화
              │
         ┌────┴────┐
      3일째     7일째
    Lv.2        Lv.3
  표현 분석   첫 주간 리포트
              │
         ┌────┴────┐
      14일째+대화3회    21일+대화5회
    Lv.4              Lv.5
  심화 리포트        ★ 심층 분석 ★
                        │
                  (문제 감지 시)
                        │
                전문가 상담 안내
```

**잠금 해제 UX:** 미해제 기능은 숨기지 않음. 탭/버튼을 누르면 "기록 OO일 + AI 대화 OO회 달성 시 이용 가능" 안내 + 진행률 바를 보여줌. 기능의 존재를 알려서 궁금증과 사용 동기를 유발.

## 심리 분석 기법 (3가지)

### 1. 감정 에너지 맵
감정을 "에너지 높음/낮음" × "기분 좋음/나쁨" 2축으로 매핑.
매일 기록에서 AI가 자동 분류해서 주간 리포트에 시각화.
매주 점이 어디에 찍히는지 보면 변화가 한눈에 보임.

- 적용: Lv.3부터 주간 리포트에 포함
- 이유: 시각적이고 직관적. 매주 변화를 차트로 보여줄 수 있어서 데모 임팩트 높음

### 2. 감정 어휘력 분석
기록에서 사용하는 감정 단어의 다양성 측정.
"힘들다"만 반복하는지, "억울하다/서운하다/지친다"를 구분하는지.
감정을 세밀하게 인식하는 것 자체가 심리학에서 중요한 스킬.

- 적용: Lv.2부터 리포트에 포함
- 이유: 문제 없는 사람도 재미있어하는 콘텐츠. "내 감정 어휘력이 12개래" → 공유 욕구

### 3. 번아웃 단계 체크
기록의 변화 추이로 현재 번아웃 단계 추정 (5단계).
열정기 → 축적기 → 만성 스트레스 → 번아웃 → 탈진.

- 적용: Lv.4부터 심화 리포트에 포함
- 이유: 타겟(직장인)에게 직접적으로 와닿음. "나 번아웃인가?" 궁금한 사람 많음

## 화면 구성 (4개 + 설정)

### ① 홈 (기록)

**첫 실행 — 초기 프로필 설문 (10~15개)**

질문 영역:
- 감정 인식: "슬플 때 보통 어떻게 반응해요?"
- 스트레스 대처: "압박감 느끼면 주로?"
- 대인관계: "갈등이 생기면 나는?"
- 자기 인식: "나의 가장 큰 강점/약점은?"
- 일상 습관: "스트레스 해소법은?"
- 에너지: "하루 중 가장 힘든 시간대는?"
- 목표: "이 앱에서 뭘 얻고 싶어요?"

→ 결과: Lv.1 기본 프로필 리포트

**매일 — 기록 화면**
- "오늘 어땠어요?" 자유 텍스트
- 음성 입력 버튼 (P1)
- 잠금 해제 진행률 바 ("심층 분석까지 5일 남았어요!")

### ② AI 대화

**공감 모드**
- 들어주기 중심, 해결책 안 줌
- "그랬구나. 힘들었겠다."

**코칭 모드**
- 리프레이밍 질문
- "정말 '항상' 그래요? 최근에 잘 된 일은 없었나요?"
- 초기 프로필 결과를 참고해서 개인화된 코칭

**오늘의 질문 (P2)**
- 심리학 기반 셀프 리플렉션
- "오늘 가장 감사했던 순간은?"

### ③ 리포트

단계에 따라 내용이 달라짐.

**Lv.1 — 초기 프로필 리포트 (가입 직후)**

```
📊 나의 마음 프로필

🧠 감정 처리 스타일: 내면화형
  → 감정을 밖으로 표현하기보다 안에서 소화하는 경향

⚡ 스트레스 반응: 회피형
  → 부담될 때 일단 피하고 보는 패턴

💬 대인관계: 조화 추구형
  → 갈등을 피하려 자기 의견을 숨기는 경향

🌱 맞춤 추천:
  → AI 대화 "코칭 모드"로 감정 표현 연습을 추천해요
```

**Lv.2 — 표현 분석 (3일차~)**

```
💬 자주 쓰는 표현
  "항상" 4회 · "또" 3회 · "어차피" 2회

📝 감정 어휘력: 8단어
  😫 힘들다 (6회) · 😤 짜증 (3회) · 😢 슬프다 (2회)

  💡 감정을 더 구체적으로 표현해보면
  자기 이해가 깊어져요!
```

**Lv.3 — 첫 주간 리포트 (7일차~)**

```
📊 첫 주간 리포트

💬 이번 주 감정 흐름
  "월~수는 안정적이었지만 목요일부터 지친 표현이 늘었어요"

🔍 자주 쓴 표현
  "항상" 4회 · "또" 3회 · "피곤" 5회

📝 감정 어휘력: 12단어 (↑4)

🗺️ 감정 에너지 맵
  이번 주는 "에너지 낮음 + 기분 나쁨" 영역에
  기록이 몰려 있어요.

🌱 긍정 포인트
  "수요일에 '괜찮았다'는 표현을 처음 쓰셨어요"

  ────────────────
  💡 다음 주도 기록하면 변화를 비교해드릴게요!
```

**Lv.4 — 심화 주간 리포트 (14일차~)**

Lv.3의 모든 것 + 비교 분석 + 패턴 해석 + 번아웃 체크 + 맞춤 제안

```
📊 2주차 심화 리포트

━━━ 이번 주 요약 ━━━

💬 감정 흐름
  "이번 주는 전반적으로 에너지가 낮았어요.
   특히 화요일과 목요일에 자책 표현이 집중되어 있어요."

🔍 자주 쓴 표현
  "또" 6회(↑) · "항상" 3회(↓) · "어차피" 4회(NEW)

📝 감정 어휘력: 15단어 (↑3)
  새로 등장: "억울하다", "답답하다", "뿌듯하다"

━━━ 지난 주와 비교 ━━━

🔄 변화 카드

  지난 주: "실수했는데 뭐 괜찮아"
  이번 주: "또 실수했다. 나는 항상 이런 식이야"

  → 같은 상황인데 자책이 강해지고 있어요.
    최근에 유독 힘든 일이 있었을까요?

🗺️ 감정 에너지 맵 비교
  지난 주: 중앙~오른쪽 위 (보통~활기)
  이번 주: 왼쪽 아래로 이동 (지침~우울)

📊 주간 비교 수치
  부정 표현: 8회 → 14회 (↑75%)
  긍정 표현: 5회 → 3회 (↓40%)

━━━ 패턴 해석 ━━━

🧠 감지된 사고 습관

  1. 확대 해석
     "안 좋은 일이 생기면 '최악의 결과'를 먼저
      떠올리는 패턴이 이번 주 3번 있었어요"
     → 지난 주(1번)보다 늘었어요

  2. 자기 일반화
     "'한 번 실수 = 나는 항상 이래'로 넓혀서
      생각하는 경향이 보여요"

━━━ 번아웃 체크 ━━━

🔥 현재 추정 단계: 2단계 (스트레스 축적기)

  1단계 열정기      ████░░░░░░ ✓ 지남
  2단계 축적기      ██████░░░░ ← 여기
  3단계 만성 스트레스 ░░░░░░░░░░
  4단계 번아웃      ░░░░░░░░░░
  5단계 탈진        ░░░░░░░░░░

  근거: "피곤" 표현 주 8회, 성취감 표현 감소 추세

  💡 지금이 관리하기 가장 좋은 타이밍이에요!

━━━ 맞춤 제안 ━━━

💡 이번 주 해볼 만한 것

  1. "또"라는 말이 나올 때 → "이번에는"으로 바꿔보기
  2. AI 대화 코칭 모드에서 리프레이밍 연습 추천
  3. 목요일마다 에너지가 떨어지네요 →
     목요일 저녁 짧은 기록 루틴 만들어보세요

  ────────────────
  💡 다음 주에 심층 분석이 해제돼요! (Lv.5까지 7일)
```

**Lv.5 — ★ 심층 분석 리포트 ★ (21일차 + AI 대화 5회)**

초기 프로필과는 차원이 다른 종합 분석.

```
📊 3주간 심층 분석 리포트

━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 종합 평가
  3주간 총 18회 기록, AI 대화 7회 분석 결과입니다.

━━━━━━━━━━━━━━━━━━━━━━━━━━

🧠 사고 패턴 심층 분석

  1. 자기비하 패턴 (빈도: 높음)
     "나는 항상 이런 식이야", "또 망했다"
     → 단발 사건을 자기 전체로 확장하는 경향
     → 특히 업무 실수 상황에서 집중 발생

  2. 미래 불안 패턴 (빈도: 중간)
     "이러다 다 망할 거야", "잘릴 것 같아"
     → 현재 상황을 최악의 결과로 확대하는 경향
     → 주로 일요일 밤~월요일에 집중

  3. 비교/열등감 (빈도: 낮음)
     "다른 사람들은 다 잘하는데"
     → 대인 접촉 후 발생

━━━━━━━━━━━━━━━━━━━━━━━━━━

📈 3주 변화 흐름

  1주차: 가벼운 스트레스, 회복력 양호
  2주차: 자책 표현 급증 (+150%), 수면 언급 증가
  3주차: 소폭 회복, 긍정 표현 등장

  [차트: 주별 감정 에너지 맵 변화]
  [차트: 부정/긍정 표현 비율 추이]

━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 감정 어휘력 성장

  1주차: 8단어
  2주차: 12단어
  3주차: 18단어

  → 감정을 점점 더 세밀하게 표현하고 있어요!
  특히 "억울하다", "뿌듯하다" 같은 구체적 표현이 늘었어요.

━━━━━━━━━━━━━━━━━━━━━━━━━━

🔥 번아웃 추이

  1주차: 1단계 (열정기)
  2주차: 2단계 (축적기) ↑
  3주차: 2단계 (축적기, 소폭 회복)

  → 2주차에 급격히 올랐지만 3주차에 안정화되고 있어요.

━━━━━━━━━━━━━━━━━━━━━━━━━━

🔬 초기 프로필과 비교

  설문 답변: "스트레스에 비교적 강한 편"
  실제 기록: 스트레스 상황에서 회피+자책 패턴 반복

  → 본인이 인식하는 것보다 스트레스 영향이
    클 수 있어요. 이 차이 자체가 중요한 발견이에요.

  설문 답변: "감정 표현이 어렵다"
  실제 기록: AI 대화에서는 꽤 솔직하게 표현하고 있어요!

  → 글로 쓸 때가 말할 때보다 표현이 자유로운 타입일 수 있어요.

━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 종합 맞춤 전략

  1. "항상", "절대" 같은 표현이 나올 때
     → 코칭 모드에서 리프레이밍 연습 추천

  2. 일요일 밤 불안이 반복되고 있어요
     → 일요일 저녁 짧은 기록 루틴을 만들어보세요

  3. 대인 접촉 후 비교감이 올라오는 패턴
     → 만남 직후 "좋았던 점 하나"를 기록하는 습관 추천

  4. 감정 어휘력이 꾸준히 성장 중!
     → 이 흐름을 유지하면 자기 이해가 더 깊어질 거예요

━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ 참고 안내

  2주차에 자책 표현이 급격히 늘었던 구간이 있었어요.
  지금은 회복세이지만, 비슷한 패턴이 반복된다면
  전문 상담사와 이야기해보는 것도 좋은 방법이에요.

  📞 정신건강위기상담전화 1577-0199
  💬 자살예방상담전화 1393

  ※ 이 분석은 AI가 생성한 참고자료이며,
  전문적인 심리 진단을 대체하지 않습니다.
```

## 개발 우선순위

### P0 (데모 필수)
1. 초기 프로필 설문 (10~15개) + Lv.1 리포트
2. 매일 기록 + 저장
3. 단계별 잠금 해제 시스템 + 진행률 바
4. Lv.3 주간 리포트 (감정 에너지 맵 + 감정 어휘력)
5. Lv.4 심화 리포트 (변화 카드 + 번아웃 체크 + 맞춤 제안)
6. Lv.5 심층 분석 리포트 (초기 프로필 비교 + 전문가 안내)
7. 테스트 모드 + 데모 데이터 3주치

### P1 (데모 퀄리티)
8. AI 대화 (공감/코칭)
9. 감정 에너지 맵 차트 시각화
10. 음성 입력

### P2 (시간 남으면)
11. 오늘의 질문
12. 다크모드
13. 데이터 내보내기

## 데모 시나리오 (5분)

### 1분 — 소개 + 초기 설문
> "감정 앱은 많죠. 파인드마이와이는 한 번 분석하고 끝이에요."
> "마음코치는 쓸수록 분석이 깊어집니다."
> → 초기 설문 시연 → Lv.1 프로필 리포트

### 1분 — 단계별 시스템 설명
> "3일 쓰면 표현 분석, 7일이면 주간 리포트..."
> → 데모 데이터로 Lv.2, Lv.3 빠르게 보여줌

### 1.5분 — Lv.4 심화 리포트 (하이라이트 1)
> "2주 쓰니까 이렇게 달라져요"
> → 변화 카드 + 번아웃 체크 시연

### 1분 — Lv.5 심층 분석 (하이라이트 2)
> "3주 쓰니까 초기 설문과 실제 기록을 비교 분석해줘요"
> → 초기 프로필 vs 실제 패턴 차이 시연
> → 전문가 안내 카드

### 0.5분 — AI 대화
> 코칭 모드에서 리프레이밍 짧게 시연
